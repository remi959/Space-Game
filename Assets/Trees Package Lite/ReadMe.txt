- Compatible with Built-in, URP, and HDRP pipelines.

 	-How to Use-

1. Import the package into your Unity project.
2. Navigate to the `Assets/Trees Package Lite` folder.
3. Drag and drop the desired prefabs into your scene.
4. If your project uses URP or HDRP, you can upgrade materials by going to:
   `Edit > Render Pipeline > URP/HDRP > Upgrade Project Materials to...`
   The package also includes preconfigured material versions for URP and HDRP.
